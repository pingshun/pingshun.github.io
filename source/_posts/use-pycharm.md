PyCharm的一些常用快捷键随手记

<!-- more -->

***编辑***
⌘ + 鼠标放在代码上         显示代码简要信息
⌘ + ⌥ + L                格式化代码
⌘ + /                    注释/取消注释与行注释
⌘ + ⌥ + /                注释/取消注释与块注释
⌃ + ⌥ + O                优化import

***查询&替换***
Double ⇧                 查询任何东西
⌘ + F                    文件内查找
⌘ + R                    文件内替换
⌘ + ⇧ + F                全局查找（根据路径）
⌘ + ⇧ + R                全局替换（根据路径）

***导航***
⌘ + ⌥ + ←                退回到上一个操作的地方
⌘ + ⌥ + →                前进到上一个操作的地方
⌘ + B / ⌘ 鼠标点击        进入光标所在的方法/变量的接口或是定义处
